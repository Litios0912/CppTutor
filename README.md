Aplicación Android para aprender C++ desde cero. Incluye editor de código, lecciones interactivas y quizzes.

## ✨ Características

- 📖 Lecciones interactivas de C++
- 💻 Editor de código integrado
- ✅ Sistema de quizzes y ejercicios
- 🏗️ Compilador server-side
- 📱 App Android nativa

## 🚀 Requisitos

- Android SDK
- Java JDK 17+
- Android Studio (recomendado)

## 🔧 Compilar

```bash
chmod +x build.sh
./build.sh
```
